Introduction 

Adapter Code for TIBCO(R) API Exchange and Joomla! enables the
creation of Joomla!-based API management portals with TIBCO(R)	
API Exchange to provide a highly branded and customizable
experience to application developers and API providers.

For more details, Please refer to the instruction.txt file at the below path 
/adapter/Examples/instructions.txt
